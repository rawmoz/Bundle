# Bundle

A macOS overlay utility for organizing files into named bundles on your desktop.

Click the menu bar icon to create a bundle — give it a name and pick a grid size
(anything from 1×1 up to 5×5). Bundles appear as floating frosted-glass panels.

Press `⌘⌥B` to show or hide all your bundles at once.

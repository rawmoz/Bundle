# Bundle

A lightweight macOS utility that gives you a temporary shelf for files while you work.

Grab a file, hit spacebar mid-drag, and Bundle catches it. Come back later, 
open the shelf, and drag it back out — exactly where you need it.

## How it works

1. Start dragging a file or folder
2. Press spacebar — Bundle's shelf slides in from the side
3. Drop the file onto it
4. Hit the global hotkey anytime to open the shelf and retrieve it
